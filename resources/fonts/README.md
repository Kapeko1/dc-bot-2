# Fonts Directory

Place the Arial Bold font file (`ARIALBD.TTF`) in this directory for inventory image generation.

You can download the font from your system:
- Windows: `C:\Windows\Fonts\arialbd.ttf`
- macOS: `/System/Library/Fonts/Supplemental/Arial Bold.ttf`

Alternatively, you can use any TrueType font (.ttf) file. Just update the font path in `app/Services/InventoryImageService.php`.
